## Development Commands

```bash
# Start development server
npm run dev

# Build for production  
npm run build

# Run tests
npm test
npm run test:e2e

# Linting and type checking
npm run lint
npm run typecheck
```

## Firebase Commands

```bash
# Deploy functions
firebase deploy --only functions

# Deploy hosting
firebase deploy --only hosting

# Local emulator
firebase emulators:start
```
